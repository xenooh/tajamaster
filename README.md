# 📝 영단어 타자 연습

## 📖 소개
영단어 연습을 위한 타자 연습 프로그램입니다.  
**초등**, **중등**, **고등** 학교 수준의 **핵심 단어**를 기준으로 구성되어 있습니다.

✅ 한글 발음 제공  
✅ 발음 기호(IPA) 표시  
✅ 단어 뜻 표시  
✅ 음성 지원 (TTS: Text-to-Speech)

---

## 🎯 주요 기능
- 🔤 초·중·고 수준별 단어 연습
- ✍️ 입력한 단어 자동 체크
- 🔎 단어의 한글 발음과 뜻 확인
- 🗣️ 발음 기호(IPA) 표시 및 음성 듣기
- 📈 점수 및 정확도 측정 (추후 확장 가능)
